Phonegap Architecture

Setup Dev Environment

  • Install Android ADT Bundle - IDE + SDK
  • Install Latest Xcode on Macintosh
  • Download NodeJS binary from nodejs.org
  • Install Cordova from CLI - npm install -g cordova

Create a New Project

  • $ cordova create hello com.example.hello "HelloWorld"
  • $ cd hello
  • $ cordova platform add android
  • $ cordova build

Integrating Your Web Code

  • In your index.html please include cordova.js script
  • <script type="text/javascript" src="cordova.js">
  • Copy your Web Code into the www folder
    inside your Cordova Project folder
  • Execute command: "cordova build"

Your app build is ready.
